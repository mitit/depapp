Перед запуском проекта стоит удостовериться, что на вашей машине установлен JDK 1.8

Запустить проект можно либо, запустив файл dep.bat, либо в командной строке перейти в корень проекта и выполнить команду: java -jar target/depapp-0.0.1-SNAPSHOT.jar <код команды> <путь к файлу с расширением xml, начиная из корня проекта>

Код команды может быть либо create (в таком случае произойдет выгрузка базы данных в xml файл), либо sync(в этом случае произойдет синхронизация содержимого таблицы БД по заданному XML файлу)

Если код команды будет некорректным или будет отсутствовать, вы получите соответствующее сообщение

Пример ввода: java -jar target/depapp-0.0.1-SNAPSHOT.jar sync dep.xml
